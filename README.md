CVify is a simple JavaFX application that helps users create their own CV.
The user can fill a form with personal information, education, skills, and experiences.
The app then shows a live preview of the CV so users can see how it will look.

**Features**

Fill up a form with:

Name

Address

Email

Phone

Education details

Skills

Work experience

Live CV preview panel

Add multiple education and experience sections

Upload a profile picture

Clean and simple JavaFX UI

Built using FXML and SceneBuilder
